More formatting

This is kind of annoying to be honest. Thankfully, this whole page is designed for anyone to hit F12 and be able to see what each section looks like. Here's an example: If you F12 and look at the Lists section, you'll see that it's filled with <ul> and <li>. That's because those are list elements.

Now let's do line breaks. Line breaks are done with <br>. Here's an example:

A creak
If only
I could close my eyes
and wake up once more
on that lonesome boulevard
to start over
in the small infinity of

A creak.


The same thing could actually be done with <pre> which preserves (see what I did there?) formatting and whitespace: 

      If only
      I could close my eyes
      and wake up once more
      on that lonesome boulevard
      to start over
      in the small infinity
                of
      A creak.

Subscripts, superscripts, etc.

As we all know, water is often represented as H2O. To subscript the 2, we use <sub>. Replace with <sup> and you'll get a superscript instead.

For example: H2O, NaHCO3 vs x2 + 4x, z3
But if we're looking into putting a ton of math equations in, we're better off with MathML or Math Markup Language.

Attributes

Attributes are things you can put into a tag to change its, well, attributes. For example, the time tag uses the datetime attribute to set the current time. Let's look at more attributes, namely global ones that are used to link HTML with CSS.

The class attribute

Targets all elements with that class in CSS or Javascript, basically putting a nametag on them. Example below:

Hi, this is an example!

The id attribute

Similar to class but only allowed to use one unique element with that id. It's really specific so avoid it if possible.

This is ligma.

Other common attributes

You can use lang to specify the language, and dir to specify the direction of the text. For example, this list below:

  • Bonjour! Salut!
  • This flows from left to right
